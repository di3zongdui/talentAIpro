📚 Candidates API
GET /api/v1/candidates 获取候选人列表
POST /api/v1/candidates 创建候选人
GET /api/v1/candidates/{id} 获取候选人详情
PUT /api/v1/candidates/{id} 更新候选人
DEL /api/v1/candidates/{id} 删除候选人
💼 Jobs API
GET /api/v1/jobs 获取职位列表
POST /api/v1/jobs 创建职位
GET /api/v1/jobs/{id} 获取职位详情
POST /api/v1/jobs/{id}/match 智能匹配
🤝 Matching API
POST /api/v1/matching/candidate/{id} 候选人匹配
POST /api/v1/matching/job/{id} 职位匹配
POST /api/v1/matching/batch 批量匹配
📊 Analytics API
GET /api/v1/analytics/pipeline 漏斗统计
GET /api/v1/analytics/performance 绩效分析
GET /api/v1/analytics/trends 趋势分析
